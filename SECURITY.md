# Security Policy

## Supported Versions

Este suportată doar versiunea curentă din repository.

| Version | Supported |
|--------|-----------|
| main / latest | ✅ |
| orice versiune veche | ❌ |

## Reporting a Vulnerability

Vulnerabilitățile sunt gestionate intern.

Pași de urmat când apare o problemă de securitate

• identificarea sursei problemei  
• evaluarea impactului  
• aplicarea remedierii  
• testare  
• commit și deploy  

Vulnerabilitățile nu se raportează public.
